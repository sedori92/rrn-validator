# 세무사무실 특급 AI 에이전트

## 프로젝트 소개

세무사무실의 반복적인 문서 업무를 AI로 자동화하기 위한 프로젝트입니다.

기한에 맞추어 부가가치세 신고, 원천징수이행상황 신고 등 세무신고를 위해 수임업체의 자료를 수집하여 세무사랑 이나 더존 회계 프로그램에 입력하기에 앞서 효율적인 전처리 과정입니다.  

사진, 이미지, PDF 및 직접 입력된 데이터를 분석하여 번호를 검증하고 문서를 분류하며 필요한 정보를 자동 추출하는 것을 목표로 합니다.

## 주요 기능

- 주민등록번호 검증
- 사업자등록번호 검증(예정)
- 법인등록번호 검증(예정)
- 카드번호 검증(예정)
- 계좌번호 검증(예정)
